Cách cài đặt theo từng bước:
B1: Tạo thư mục nơi chứa đồ án và dùng terminal để chạy: pip install virtualenv 
    sau đó chạy			  		       : virtualenv env (để tạo thư mục env) 

B2: Sử dụng terminal tiếp tục chạy: .\env\Scripts\activate để chạy trên môi trường ảo.

B3: Đưa toàn bộ các thư mục trừ env sang thư mục đang sử dụng.

B4: Sử dụng terminal cài đặt thư viện: pip install Flask
				       pip install chatterbot
				       pip install chatterbot-corpus 
 
B5: Sử dụng terminal chạy lệnh: python app.py để chạy chương trình
